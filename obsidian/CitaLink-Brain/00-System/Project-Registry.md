## Add A Project

1. Copy [[_Project-Template]] into `Projects/project-slug/00-Project-Index.md`.
2. Define objective, owner, current status, and next action.
3. Choose safe context level.
4. Create an external AI policy.
5. Add the project to this registry.

## Project Health Rules

- Every active project must have one current next action.
- Every active project must have one owner.
- Every active project must have a safe context level.
- Every external AI handoff must point back to a project.
