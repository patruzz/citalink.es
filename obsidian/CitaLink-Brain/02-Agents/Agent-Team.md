# Agent Team

## Meeting Format

Use this note as the agenda for agent conversations.

1. Captain Agent states the weekly objective.
2. Market Scout Agent brings accounts and evidence.
3. Offer Agent turns evidence into message angles.
4. Product Agent maps promises to a pilot workflow.
5. Compliance Agent checks risk and consent.
6. Captain Agent assigns the next action.

## Current Weekly Objective

Book the first CitaLink appointment audit with a business that depends on consultations, valuations, visits, or first meetings.

## Agent Briefs

### Captain Agent

Question to answer:

> What single action most increases the probability of first customer this week?

Output:

- One daily priority.
- One owner.
- One next action.

### Market Scout Agent

Question to answer:

> Which accounts visibly lose value when they respond slowly?

Output:

- Account name.
- Website.
- Sector.
- Appointment channel.
- Leakage hypothesis.
- Score.

### Offer Agent

Question to answer:

> What message would make this specific decision maker curious enough to answer?

Output:

- Opening line.
- Pain hypothesis.
- Audit CTA.
- Follow-up.

### Product Agent

Question to answer:

> What is the smallest workflow we can activate without overwhelming the customer?

Output:

- Source channel.
- Qualification questions.
- Calendar rule.
- Human review rule.
- Dashboard metric.

### Compliance Agent

Question to answer:

> Are we respecting consent, preferences, and reasonable contact boundaries?

Output:

- Approved, revise, or block.
- Required opt-out language.
- Data handling note.
